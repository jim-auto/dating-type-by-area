# Dating Type by Area

Dating behavior patterns vary by neighborhood. This project visualizes how local culture, resident demographics, and social infrastructure shape dating behavior into two types: **fast dating** (quick relationship progression) and **slow dating** (gradual relationship progression).

## Structure

```
Street Culture → Resident Type → Dating Behavior Type
```

## Features

- Interactive map of Japan (Tokyo, Osaka, Nagoya) powered by Leaflet.js
- Area cultural analysis with traits and resident profiles
- Dating type classification (fast / slow) with explanations
- Dark mode, minimal GitHub-style UI

## How it works

Click any area marker on the map to see:

- **Area Traits** — cultural characteristics of the neighborhood
- **Typical Residents** — who lives there
- **Dating Type** — fast or slow
- **Explanation** — why that area has that dating pattern

## Tech stack

HTML, CSS, Vanilla JavaScript, Leaflet.js, GitHub Pages
